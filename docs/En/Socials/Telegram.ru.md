# Документация класса Telegram

## Класс: Telegram

### Конструктор

#### Telegram(IZennoPosterProjectModel project, bool log = false)

**Назначение**: Инициализирует новый экземпляр бота Telegram для отправки уведомлений и отчетов.

**Пример**:
```csharp
var telegram = new Telegram(project, log: true);
```

**Разбор**:
```csharp
// Параметры:
// - project: IZennoPosterProjectModel - Модель проекта для доступа к переменным и логированию
// - log: bool - Включить детальное логирование (по умолчанию: false)
// Возвращает: Экземпляр Telegram
// Примечание: Автоматически загружает токен бота, ID группы и ID темы из переменных проекта
// Не требует параметр Instance (использует HTTP API вместо автоматизации браузера)
```

---

## Методы уведомлений

### Report()

**Назначение**: Отправляет отчет о выполнении в группу Telegram с деталями успеха или неудачи.

**Пример**:
```csharp
telegram.Report();
```

**Разбор**:
```csharp
// Параметры: Отсутствуют
// Возвращает: void
// Примечание: Проверяет переменную проекта "failReport" на наличие сообщения об ошибке
// Если failReport пуста, отправляет сообщение об успехе с именем проекта и информацией об аккаунте
// Отправляет форматированное сообщение в указанную группу/тему Telegram
// Использует форматирование MarkdownV2 для текста сообщения
// Логирует финальное время выполнения и статус в лог ZennoPoster с цветами (зеленый для успеха, оранжевый для неудачи)
```
